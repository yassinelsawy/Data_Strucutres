# Sorting and Data Structures Project

This project includes implementations of various sorting algorithms, data structures, and additional functionalities in C++.

## Features

### Sorting Algorithms

1. **Insertion Sort**
2. **Selection Sort**
3. **Bubble Sort**
4. **Merge Sort**
5. **Quick Sort**

### Data Structures

1. **Stack (Array-based)**
2. **Stack (Linked List-based)**
3. **Linked List**
